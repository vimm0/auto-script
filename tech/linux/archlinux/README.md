## Ssh setup
- `ssh-keygen -t rsa -b 4096 -C "your_email@example.com"`
- `eval "$(ssh-agent -s)"`
- `ssh-add ~/.ssh/id_rsa`
- `sudo pacman -S xclip`
- `xclip -sel clip < ~/.ssh/id_rsa.pub`
- paste public key to Git repositories
